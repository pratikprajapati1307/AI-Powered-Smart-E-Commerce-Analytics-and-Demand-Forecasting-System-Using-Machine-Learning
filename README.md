# 📊 AI-Powered E-Commerce Analytics & Demand Forecasting System

A simple and powerful AI-based web application that analyzes e-commerce data and provides smart business insights using Machine Learning.

---

## 🚀 Features

- 📈 Demand Forecasting  
- 🚨 Stockout & Overstock Detection  
- 💰 Revenue Prediction  
- 📊 Sales Trend Analysis  
- 🏆 Top Selling Products  
- 📅 Monthly / Seasonal Demand  
- 👥 Customer Insights  

---

## ⚙️ Technologies Used

- Backend: Flask (Python)  
- Frontend: HTML, Tailwind CSS, JavaScript  
- ML Models: RandomForest, Linear Regression  
- Libraries: Pandas, NumPy, Scikit-learn  

---

## ▶️ How to Run

1. Install dependencies:

pip install -r requirements.txt

2. Run the application:

python app.py

3. Open in browser:

http://127.0.0.1:5000

---

## 📊 Dataset

- system_data.csv → Training data  
- sample_upload.csv → Demo/testing data  

---

## 🧠 How It Works

1. Upload CSV file  
2. System detects columns automatically  
3. Data is cleaned and processed  
4. ML model predicts future demand  
5. Dashboard shows insights  
6. Alerts suggest business actions  

---

## 🎯 Purpose

This project helps businesses to:

- Predict future demand  
- Avoid stockout & overstock  
- Improve sales planning  
- Make data-driven decisions
